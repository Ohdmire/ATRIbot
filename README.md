## MongoDB储存的数据

- user表
储存基本信息和~~bp1-100的各项信息(array)~~ 现已分为bp表

- bp表
储存格式为列表

- score表
储存用户在谱面上的记录 精简score，去除无用数据

- bind表
储存用户的qqid

-group表
储存群的成员列表
